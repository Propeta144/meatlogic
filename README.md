# MeatLogic: Integrated Order & Manual Stock Management System

A web-based ordering and stock management system built for Arnel's Karne atbp.

**Live Hosted Link:** [https://meatlogic.vercel.app/](https://meatlogic.vercel.app/)

---

## 🚀 Setup Instructions

This system uses Firebase for its database. The following steps are for the administrator to set up the shop for the first time after deployment.

### Step 1: Access the Admin Panel

1.  Go to the live website: [https://meatlogic.vercel.app/](https://meatlogic.vercel.app/)
2.  Scroll to the bottom of the page (footer).
3.  Click the link labeled **`🔒 Admin Access`**.

### Step 2: Log In as Admin

Use the default administrator credentials to log in:
*   **Email:** `adminrhon@meatlogic.com`
*   **Password:** `adminrhon123`

### Step 3: Load the Products (First-Time Use Only)

1.  After logging in, you will land on the **Dashboard**.
2.  Find and click the button labeled **`🚀 Load Default Products`**.
3.  This will populate the database with a sample list of meat products (beef, pork, chicken, etc.). This button will disappear once products exist in the database.

### Step 4: Set the Daily Stock

1.  In the sidebar, navigate to **`📦 Stock Management`**.
2.  For each product, enter the quantity you want to allocate for online sales **for that day**.
3.  After setting the quantities, click the **`💾 Save All Changes`** button.

**The setup is complete!** The online shop is now ready to accept orders.
